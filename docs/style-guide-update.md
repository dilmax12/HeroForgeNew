# Atualização do Guia de Estilo

## Botões
- Primário: `buttonPrimary` — gradiente sazonal, `px-3 py-2`, foco com `ring-2`, transição `duration-200`
- Secundário: `buttonSecondary` — cinza neutro, `px-2 py-1`, texto `xs`, transição `duration-200`

## Containers
- Seções com `transition-all duration-200` para transições suaves

## Acessibilidade
- `aria-live="polite"` para mensagens de aviso
- `aria-expanded` em toggles
- Estados de foco consistentes com `focus:ring` e `focus:ring-offset-1`

## Layout
- Grids responsivos: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`
- Agrupamento de ações primárias acima das sinergias
